1. 单节点挂载磁盘： single_format_disk.sh
    1.1. 安装注意事项：
		1.1.1. 不能选择系统盘所在磁盘！！！！！！！
        1.1.2. 只适合单块磁盘只创建一个分区的情况！！！！！
	1.2. 使用前修改以下参数
        1.2.1. DEVICE_LIST 需要格式化的磁盘列表
        1.2.2. dir         需要mount的起始路径，如：第一块盘mount在/mnt/disk1  则dir=1 ；后面按顺序递增


2. 一键多节点挂载磁盘：batch_mount_disk.sh    依赖mount.sh 文件
    2.1. 安装注意事项：
        2.1.1. 不能选择系统盘所在磁盘！！！！！！！
        2.1.2. 只适合单块磁盘只创建一个分区的情况！！！！！
        2.1.3. 如果保留免密登录   请把第53行注释掉
        2.1.4. 使用时，把 mount.sh 与 batch_mount_disk.sh 放在同一个路径下
		2.1.5. 如果是初次安装，没有执行过ssh命令连接到其他主机上，doExpect() 注释掉第二行，解注释第一行
		       如果执行过ssh命令，doExpect()函数函数不用修改
	2.2. 使用前修改以下参数
        2.2.1. cluster_ip    集群所有节点的ip
        2.2.1. passwd        root用户登录密码
        2.2.1. DEVICE_LIST   需要格式化的磁盘列表
        2.2.1. dir           需要mount的起始路径，如：第一块盘mount在/mnt/disk1  则dir=1 ；后面按顺序递增
